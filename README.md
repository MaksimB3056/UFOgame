# 🛸 UFO Game (Unity)

A simple 3D arcade-style game developed in Unity.  
The project demonstrates basic player movement, object collection mechanics, and simple game logic.

---

## 📌 Overview

This is a lightweight Unity game where the player controls a UFO and collects objects within the game world.

The project was created as a learning exercise to practice:
- Player movement and controls
- Collision detection
- Object interaction and collection system
- Basic game loop design
- Scene management in Unity

---

## 🎮 Gameplay Features

- 🛸 Player-controlled UFO
- ⭐ Collectible objects scattered around the map
- 💥 Collision detection system
- 🧭 Simple level navigation
- 🏁 Basic win/finish condition (if implemented)
- 🔁 Restart / replay functionality

---

## 🛠️ Built With

- Unity Engine
- C#
